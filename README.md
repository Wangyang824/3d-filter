# 3D彩票筛选器

一个简单的手机友好网页工具，用于筛选 `000-999` 号码。

## 已实现功能

- 排除某些数字（如 `1,4,9`）
- 只保留含某些数字的号码（如 `0,3,8`）
- 排除和值（如 `6,9,15`）
- 和值范围筛选（最小值/最大值）
- 跨度范围筛选（最小值/最大值）
- 形态筛选（豹子/对子/组六）
- 排除对子（恰好两位相同）
- 排除豹子（三位相同）
- 胆码（至少命中 1 个）
- 杀码（命中即排除）
- 奇数个数筛选
- 大数个数筛选（5-9 为大数）
- 012 路筛选
- 定位筛选（百位/十位/个位保留与排除）

## 使用方法

直接用浏览器打开 `index.html` 即可使用。

如需手机使用，把项目放到任意静态托管后在手机浏览器访问即可。

## PWA 安装（手机像小程序一样用）

项目已支持 PWA（可添加到手机主屏幕）。

- 必须通过 `https` 域名访问（本地 `localhost` 测试也可）
- Android Chrome：打开页面后点菜单 -> 添加到主屏幕
- iPhone Safari：点分享 -> 添加到主屏幕

首次联网打开后，页面资源会缓存，弱网或离线也可以打开基础功能。
